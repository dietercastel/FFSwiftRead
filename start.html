Prepare to speed through your browser-based reading, learn more, and save time.

If you're interested in signing up for email updates, productivity tips, and other goodies, you can do so on our website.

1. Pin the SwiftRead extension icon for easy access

2. Use SwiftRead on text. There are four ways to do so:

Extension Icon

  • 1. Open the SwiftRead menu by clicking on the SwiftRead extension icon next to the browser address bar.
  • 2. Select "SwiftRead current page".

Auto-extract, Alt+V (default hotkey)

  • Just hit Alt+V to automatically extract the main content of a webpage and open SwiftRead!

Select Text

  • 1. Select text on the webpage with your mouse.
  • 2. Right click, then select "SwiftRead selected text" in the menu.
  • OR hit Alt+Vto open SwiftRead with the selected text.

Copy-paste Text

  • 1. Copy text from anywhere (e.g. PDFs).
  • 2. Open the SwiftRead menu by clicking on the SwiftRead extension icon next to the browser address bar.
  • 3. Select "Paste text into SwiftRead".

Now give it a shot! Try SwiftReading a news article.
