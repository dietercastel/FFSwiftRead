Here's how to use SwiftRead on Google Docs Available on PRO

1. Visit Google Docs

2. Scroll to the page in your Google Doc that you want to start reading from.

3. Open SwiftRead from menu or use hotkeys to auto-extract the text on the current page.

4. Turn to the preceding or following pages within SwiftRead by clicking on the "left" and "right" icons
